    • title:Linux常用命令

    基本概念：

    1. root表示当前登录用户
    2. localhost表示主机名 默认为localhost
    3. ~表示当前所在目录(家目录)
      root用户家目录是/root,其他用户家目录是/home/用户名/ ，比如是test用户，家目录是/home/test/,建立用户的时候，家目录自动生成，也就是说，linux保证每一个用户都有一个家。可以通过pwd命令查看当前所在目录
    4. # 超级用户提示符   $普通用户提示符

    命令格式

    1.个别命令并不是这样格式的
    2.当有多个选项时，可以写在一起
    3.简化选项与完整选项，例如 -a 等价于 —a

    一、查询目录中的内容：ls命令

    基本语言：

    选项:

    -a   显示所有文件，包括隐藏文件
    -l   显示详细信息
    -d   查看目录属性
    -h   人性化显示文件大小
    -i   显示inode 也就是id

    当输入ls -lh的时候，第一列显示内容代表权限

    权限默认10位标识，第一位代表文件类型，常用的类型如下：

    - 代表文件
    d 代表目录
    l 软链接文件（快捷方式）

    剩下九位每三位为一组，代表这个文件的所有者、所属组（用户组）、其他人三个身份的权限。
    身份：

    u 所有者
    g 所属组
    o 其他人

    权限：

    r 读
    w 写
    x 执行

    通过知识讲解，分析一下：

    1.开始第一个字符是-说明这是个文件
    2.rw- 读写权限 root用户拥有（显示内容中第一个root）
    3.r-- 只读权限 root组拥有（注意root属于所有者，在root组属于个例，root组中其他人只读权限）
    4.r-- 其他人只读权限
    5.. 代表ACL权限

    1. 1 引用计数，代表在软链接引用次数

    二、文件处理命令

    1.目录处理命令

    1).建立目录

    基本语法：

    1.-p 选项代表递归创建，比如 mkdir -p /home/dd/dds
    2.如果目录存在，会提示已经存在，不会创建成功
    3.如果 m1和m2目录不存在,命令 mkdir /home/m1/m2会出错无法创建目录"/m1/m2": 没有那个文件或目录

    2)切换目录

    基本语法：

    1.与window系统一样
    2.cd ~或者cd进入当前登录用户家目录
    3.cd -进入上次目录
    4.cd ..进入上一级目录
    5.cd . 进入当前目录
